# BL_reporters
Scripts for data processing used in paper "Metabolically robust autoluminescent reporters"

Boxplots .ipynb contains pipeline commonly used for plotting all boxplots (with or without Conover's test heatmap).

Scatterplots.ipynb contains pipeline commonly used for plotting all scatterplots (Supplementary Figures 3-6).

Kinetics.ipynb contains pipeline commonly used for plotting lineplots (including Supplementary Figures 10-11).
